мқҙлІҲ мӢңк°„м—җлҠ” OAuth лЎңк·ёмқёмқҳ к°ңл…җкіј мӣҗлҰ¬м—җ лҢҖн•ҙ м•Ңм•„ліҙкі , ExpressмҷҖ Passport.jsлҘј мӮ¬мҡ©н•ҳм—¬ OAuth лЎңк·ёмқёмқ„ кө¬нҳ„н•ҳлҠ” л°©лІ•мқ„ мӢӨмҠөн•ҙліҙкІ мҠөлӢҲлӢӨ. OAuth лЎңк·ёмқёмқҖ нҳ„лҢҖ мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң мӮ¬мҡ©мһҗ мқёмҰқмқ„ к°„нҺён•ҳкІҢ кө¬нҳ„н•  мҲҳ мһҲлҠ” мӨ‘мҡ”н•ң кё°лҠҘ мӨ‘ н•ҳлӮҳмһ…лӢҲлӢӨ.

OAuth лЎңк·ёмқё к°ңл…җ

OAuthлһҖ?

OAuth(Open Authorization)лҠ” мӮ¬мҡ©мһҗк°Җ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ л№„л°ҖлІҲнҳёлҘј м ңкіөн•ҳм§Җ м•Ҡкі лҸ„ лӢӨлҘё м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң мһҗмӢ мқҳ кі„м • м •ліҙлҘј м•Ҳм „н•ҳкІҢ кіөмң н•  мҲҳ мһҲкІҢ н•ҳлҠ” мқёмҰқ н”„лЎңнҶ мҪңмһ…лӢҲлӢӨ. OAuthлҠ” мЈјлЎң мҶҢм…ң лЎңк·ёмқё(Google, Facebook, GitHub л“ұ)м—җм„ң мӮ¬мҡ©лҗ©лӢҲлӢӨ.

OAuthмқҳ мЈјмҡ” мҡ©м–ҙ

  1. лҰ¬мҶҢмҠӨ мҶҢмң мһҗ(Resource Owner): мӮ¬мҡ©мһҗлЎң, м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ м ‘к·јн•ҳл ӨлҠ” мӮ¬лһҢмһ…лӢҲлӢӨ.
  2. нҒҙлқјмқҙм–ёнҠё(Client): лҰ¬мҶҢмҠӨ мҶҢмң мһҗмқҳ м •ліҙлҘј мҡ”мІӯн•ҳлҠ” м• н”ҢлҰ¬мјҖмқҙм…ҳмһ…лӢҲлӢӨ.
  3. лҰ¬мҶҢмҠӨ м„ңлІ„(Resource Server): мӮ¬мҡ©мһҗ м •ліҙлҘј м ҖмһҘн•ҳкі  мһҲлҠ” м„ңлІ„мһ…лӢҲлӢӨ.
  4. мқёмҰқ м„ңлІ„(Authorization Server): мӮ¬мҡ©мһҗлҘј мқёмҰқн•ҳкі , м—‘м„ёмҠӨ нҶ нҒ°мқ„ л°ңкёүн•ҳлҠ” м„ңлІ„мһ…лӢҲлӢӨ.
  5. м—‘м„ёмҠӨ нҶ нҒ°(Access Token): мқёмҰқ м„ңлІ„м—җм„ң л°ңкёүн•ҳлҠ” нҶ нҒ°мңјлЎң, лҰ¬мҶҢмҠӨ м„ңлІ„м—җ мҡ”мІӯн•  л•Ң мӮ¬мҡ©лҗ©лӢҲлӢӨ.

OAuth лЎңк·ёмқё нқҗлҰ„

  1. мӮ¬мҡ©мһҗ мҡ”мІӯ: мӮ¬мҡ©мһҗк°Җ нҒҙлқјмқҙм–ёнҠё м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ лЎңк·ёмқё мҡ”мІӯмқ„ н•©лӢҲлӢӨ.
  2. к¶Ңн•ң л¶Җм—¬ мҡ”мІӯ: нҒҙлқјмқҙм–ёнҠё м• н”ҢлҰ¬мјҖмқҙм…ҳмқҙ мқёмҰқ м„ңлІ„м—җ к¶Ңн•ң л¶Җм—¬лҘј мҡ”мІӯн•©лӢҲлӢӨ.
  3. мӮ¬мҡ©мһҗ мқёмҰқ: мқёмҰқ м„ңлІ„к°Җ мӮ¬мҡ©мһҗлҘј мқёмҰқн•ҳкі , к¶Ңн•ң мҡ”мІӯмқ„ мҠ№мқён•©лӢҲлӢӨ.
  4. м—‘м„ёмҠӨ нҶ нҒ° л°ңкёү: мқёмҰқ м„ңлІ„к°Җ нҒҙлқјмқҙм–ёнҠё м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ м—‘м„ёмҠӨ нҶ нҒ°мқ„ л°ңкёүн•©лӢҲлӢӨ.
  5. API мҡ”мІӯ: нҒҙлқјмқҙм–ёнҠё м• н”ҢлҰ¬мјҖмқҙм…ҳмқҙ м—‘м„ёмҠӨ нҶ нҒ°мқ„ мӮ¬мҡ©н•ҳм—¬ лҰ¬мҶҢмҠӨ м„ңлІ„м—җ мӮ¬мҡ©мһҗ м •ліҙлҘј мҡ”мІӯн•©лӢҲлӢӨ.
  6. мқ‘лӢө: лҰ¬мҶҢмҠӨ м„ңлІ„к°Җ мӮ¬мҡ©мһҗ м •ліҙлҘј л°ҳнҷҳн•©лӢҲлӢӨ.

мӨҖл№„н•ҳкё°

лЁјм Җ н•„мҡ”н•ң нҢЁнӮӨм§ҖлҘј м„Өм№ҳн•©лӢҲлӢӨ:

npm init -y
npm install express passport passport-google-oauth20 express-session

м„ңлІ„ м„Өм •

лӢӨмқҢмқҖ н•„мҡ”н•ң лӘЁл“Ҳмқ„ к°Җм ёмҳӨкі  м„ңлІ„лҘј м„Өм •н•ҳлҠ” мҪ”л“ңмһ…лӢҲлӢӨ. мһҗм„ён•ң л¶Җл¶„мқҖ л’Өм—җм„ң м„ӨлӘ…н•ҳкІ мҠөлӢҲлӢӨ.

// server.js
const express = require("express");
const passport = require("passport");
const session = require("express-session");
const { Strategy: GoogleStrategy } = require("passport-google-oauth20");

const app = express();
const port = 3000;

// м„ём…ҳ м„Өм •
app.use(session({ secret: "secret", resave: false, saveUninitialized: true }));

// Passport мҙҲкё°нҷ”
app.use(passport.initialize());
app.use(passport.session());

// мӮ¬мҡ©мһҗ м§Ғл ¬нҷ” л°Ҹ м—ӯм§Ғл ¬нҷ”
passport.serializeUser((user, done) => {
  done(null, user);
});
passport.deserializeUser((obj, done) => {
  done(null, obj);
});

// Google OAuth2.0 м „лһө м„Өм •
passport.use(
  new GoogleStrategy(
    {
      clientID: process.env.GOOGLE_CLIENT_ID,
      clientSecret: process.env.GOOGLE_CLIENT_SECRET,
      callbackURL: "<http://localhost:3000/auth/google/callback>",
    },
    (token, tokenSecret, profile, done) => {
      return done(null, profile);
    }
  )
);

// лқјмҡ°нҠё м„Өм •

// нҷҲ лқјмҡ°нҠё
app.get("/", (req, res) => {
  res.send('<h1>Home</h1><a href="/auth/google">Login with Google</a>');
});

// лЎңк·ёмқё лқјмҡ°нҠё
app.get(
  "/auth/google",
  passport.authenticate("google", {
    scope: ["<https://www.googleapis.com/auth/plus.login>"],
  })
);

// мҪңл°ұ лқјмҡ°нҠё
app.get(
  "/auth/google/callback",
  passport.authenticate("google", { failureRedirect: "/" }),
  (req, res) => {
    res.redirect("/profile");
  }
);

// н”„лЎңн•„ лқјмҡ°нҠё
app.get("/profile", (req, res) => {
  if (!req.isAuthenticated()) {
    return res.redirect("/");
  }
  res.send(
    `<h1>Profile</h1><p>${req.user.displayName}</p><a href="/logout">Logout</a>`
  );
});

// лЎңк·ём•„мӣғ лқјмҡ°нҠё
app.get("/logout", (req, res) => {
  req.logout();
  res.redirect("/");
});

app.listen(port, () => {
  console.log(`Server is running on <http://localhost>:${port}`);
});
